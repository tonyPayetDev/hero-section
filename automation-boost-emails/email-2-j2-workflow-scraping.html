Le workflow en 6 nodes

01

Schedule Trigger

Lundi 8h00 — cron 0 8 * * 1. Aucune intervention manuelle.
02

HTTP Request → Google Maps API

Places API — textsearch ou nearbysearch. Query paramétrable : type de business + rayon + ville. Retourne nom, adresse, note, nb avis, téléphone, site web.
03

Code Node — Filtrage & déduplication

JS vanilla. Filtre les entrées déjà connues via Google Sheets (place_id unique). Ne garde que les nouveaux leads.
04

Google Sheets — Append rows

Stocke chaque nouveau prospect : nom, contact, note, statut = "À contacter". Base de données persistante, zéro coût.
05

Aggregate Node — Résumé hebdo

Consolide tous les nouveaux prospects de la semaine en un seul objet JSON structuré.
06

Send Email — Recap HTML

Email auto chaque lundi avec : nb nouveaux prospects, top 3 par note, lien vers le Google Sheet. Tu ouvres, tu prospectes.

Ce que ça donne en pratique

Setup : 47 min une seule fois.
Coût récurrent : quasiment 0 (Google Maps API gratuit jusqu'à 200$/mois de requêtes).
Résultat : 15 à 40 nouveaux leads qualifiés chaque lundi, sans rien faire.
